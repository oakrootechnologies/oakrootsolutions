# Video Assets

Place your hero video file here with the name `hero-video.mp4`.

## Video Requirements

- **Format**: MP4 (H.264 codec recommended)
- **Recommended dimensions**: 1920x1080 (Full HD) or 3840x2160 (4K)
- **Aspect ratio**: 16:9
- **File size**: Optimize for web (keep under 10MB if possible)
- **Duration**: Loop-friendly (10-30 seconds recommended)

## Usage

The video will auto-play, loop, and be muted for the hero section background.

## Fallback

If the video file is not found, a gradient background will be displayed instead.
